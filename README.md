# Damian Trajkovski — Portfolio

My personal portfolio website built with **Next.js 14**, **TypeScript**, **GSAP**, and **Tailwind CSS**.  
Designed to showcase my projects, experience, and background as a software engineer.

🔗 **Live site:** https://damian-t-resume-portfolio.vercel.app

---

## 🛠 Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- GSAP + ScrollTrigger
- Lenis smooth scroll
- Vercel (deployment)

---

## 📁 Overview

The site includes:

- Smooth scroll and scroll-triggered animations  
- Responsive editorial layout  
- Custom cursor (desktop)  
- Clean typography using Cormorant Garamond + Syne  

All personal content is stored in:

src/lib/data.ts
